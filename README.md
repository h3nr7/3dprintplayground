# 3dprintplayground

#### Current Setup
* Creality Ender 5 Pro
* BL Touch physical bed levelling sensor
* Raspberry Pi 4 with OctoPrint for remote monitoring
* (upcoming upgrade) Micro Swiss Hotend
* (upcoming upgrade) Zodiac MK8 Nozzle
